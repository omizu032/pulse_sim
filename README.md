# スーパーカブエンジン　パルスシミュレータ

## 概要
スーパーカブ用の自作デジタルCDIの動作検証のために、Arduino Nanoベースでクランクセンサー（ピックアップコイル）からのパルスをシミュレートするパルスジェネレータです。

一般的なアナログタイマーIC（NE555等）のRC発振で生じるフェーズノイズ（ジッター）を排除して、実際のエンジン回転に近い「滑らかな加減速」をシミュレートできます。

## 主な特徴
* **ジッターフリー**: ATmega328Pのハードウェアタイマー（Timer1）を直接駆動することで、ソフトウェアによる遅延を極力排除して精度の高いパルスを生成します。
* **マニュアルモード**: 10kΩ可変抵抗（VR）で回転数（パルス周期）を制御できます。EMA（指数移動平均）フィルタとヒステリシスにより、指先の微細なブレや電気的ノイズをカットしています。
* **スイープモード**: 指定されたRPM範囲をスロープで自動加減速します。開始方向（Max/Min）や往復回数の指定も可能です。VRを手で可変させなくても、実際の走行時に近い回転数の加減速が実現できます。
* **シリアルモニタリング**: 現在の周波数（Hz）と回転数（RPM）をシリアルモニタに出力します。

## ハードウェア構成
* **MCU**: Arduino Nano (ATmega328P / 5V / 16MHz)
* **入力デバイス**: 
  * 10kΩ 可変抵抗 (Bカーブ推奨)
  * タクトスイッチ（何でも可）
* **ピンアサイン**:
  * `A0` : 可変抵抗 (ワイパー端子)
  * `D2` : スイープ起動スイッチ (内部プルアップ使用、Active-LOW)
  * `D10`: パルス出力 (CDI基板のJ3入力へ接続)
  * `GND`: CDI基板と共通化

## 使用方法
1. 回路をブレッドボードなどを使って結線し、Arduinoに電源を投入します。
2. **通常時**: 可変抵抗を回すことで、任意の回転数（RPM）に固定できます。
3. **スイープテスト**: タクトスイッチを押すと、あらかじめ設定された条件（例: 2500 RPM ⇔ 4500 RPM）で指定回数の自動加減速テストが実行され、終了すると自動で通常時に戻ります。
