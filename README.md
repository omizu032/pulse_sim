# 高精度エンジンパルスシミュレータ (CDI開発用)

## 概要
本プロジェクトは、バイクの自作デジタルCDIの動作検証を目的とした、Arduino Nanoベースのダミーパルスジェネレータです。

CDI側に実装された「1回転先予測進角アルゴリズム」を正確に評価するため、一般的なアナログタイマーIC（NE555等）のRC発振で生じるフェーズノイズ（ジッター）を完全に排除し、実際の重いフライホイールの慣性に近い「滑らかな加減速」をデジタル空間でシミュレートします。

## 主な特徴
* **完全ジッターフリーパルス**: ソフトウェアによる遅延を排除し、ATmega328Pのハードウェアタイマー（Timer1）を直接駆動することで、クリスタル精度のパルスを生成します。
* **高精度マニュアルモード**: 10kΩ可変抵抗（VR）による回転数制御。EMA（指数移動平均）フィルタとヒステリシスにより、指先の微細なブレや電気的ノイズを完全にカットします。
* **オートマチックスイープモード**: 物理的にあり得ない加速度（過大な$dV/dt$）を排除するため、指定されたRPM範囲を数学的に完璧なスロープで自動加減速します。開始方向（Max/Min）や往復回数の指定も可能です。
* **非同期シリアルモニタリング**: パルス生成処理に一切の干渉を与えずに、現在の周波数（Hz）と回転数（RPM）をリアルタイム出力します。

## ハードウェア構成
* **MCU**: Arduino Nano (ATmega328P / 5V / 16MHz)
* **入力デバイス**: 
  * 10kΩ 可変抵抗 (Bカーブ推奨)
  * タクトスイッチ
* **ピンアサイン**:
  * `A0` : 可変抵抗 (ワイパー端子)
  * `D2` : スイープ起動スイッチ (内部プルアップ使用、Active-LOW)
  * `D10`: パルス出力 (CDI基板のJ3入力へ接続)
  * `GND`: CDI基板と共通化

## 使用方法
1. 回路を結線し、Arduinoに電源を投入します。
2. **通常時**: 可変抵抗を回すことで、任意の回転数（RPM）に固定できます。
3. **スイープテスト**: タクトスイッチを押すと、あらかじめ設定された条件（例: 2500 RPM ⇔ 4500 RPM）で指定回数の自動加減速テストが実行され、終了すると自動で通常時に戻ります。
